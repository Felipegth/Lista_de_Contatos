# Lista de Contatos
Projeto modelo de estudo de aplicação de conceitos SOLID em .NET.

## Prerequisites
*.NET 5.0
*Visual Studio Code

### Create project

'''shell
mkdir ./src
touch README.md
git init
dotnet new gitignore
'''

'''shell
cd src
dotnet new console -o Lista_de_Contatos
'''

#### Build
'''shel
dotnet run --project src\Lista_de_Contatos\Lista_de_Contatos.csproj
'''
